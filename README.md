# ✌️ `ye-quote`

Fetch a random quote from Kanye West with TypeScript.

## Installation

```bash
npm i ye-quote
# or
yarn add ye-quote
```

## Usage

An idea of how you can use this:

```ts
import { getRandomKanyeQuote } from "ye-quote";

useRandomKanyeQuote().then((quote) => {
  console.log(quote);
}.catch((err) => console.log(err));
```
